# Maharashtra RC Transfer — Physical Document Verification Guide

> Every document to ask for, every field to read, every cross-match to run — **no government website needed**  
> Companion to: Maharashtra RC Transfer Complete Branching Checklist  
> Use this guide when you have all documents on the table in front of you

---

## HOW TO USE THIS GUIDE

- Each section covers **one document** you ask the seller to produce.
- Under each document: the **exact fields** to read, **where to find them** on the physical document, and **what to match them against**.
- `→ MATCH AGAINST` = the other document/source you compare that field to.
- `🔴 RED FLAG` = if this match fails, stop the deal until resolved.
- `🟡 FLAG` = worth questioning; not necessarily a deal-breaker.
- The final section (**Master Cross-Match Matrix**) consolidates every match in one table.
- The **Physical Car Inspection** section covers checks you do on the vehicle itself — no documents needed, just a torch.

# DOCUMENT 1 — RC BOOK / RC SMART CARD

> The master document. **Every other document must agree with the RC.**  
> Ask: _"Please show me the original RC book / RC smart card."_  
> ⚠️ DigiLocker RC is NOT accepted at any Maharashtra RTO for ownership transfer. Original physical RC only.

---

## Field 1.1 — Registered Owner Name

**Where to find it:**

- RC Book: First page — _"Name of owner"_ field.
- RC Smart Card: Printed on the front face below the registration number.

| Match against                                                                          | What to check                                                                                                       | Result if mismatch                                                                                                                                            |
| -------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Seller's **Aadhaar card** — _"Name"_ field                                             | Must be character-for-character identical. No abbreviations, no nicknames, no shortened forms.                      | 🔴 RED FLAG — Stop. Ask for legal proof of name change (marriage certificate / court order / gazette notification). If no valid explanation → do not proceed. |
| Seller's **PAN card** — _"Name"_ field                                                 | PAN may use initials for middle name — cross-check with Aadhaar if a discrepancy.                                   | 🟡 FLAG — PAN format differences are common; confirm with Aadhaar as primary.                                                                                 |
| **Insurance policy** — _"Name of insured"_ / _"Proposer name"_ field                   | Should match RC owner name.                                                                                         | 🟡 FLAG — A mismatch may mean the vehicle was informally transferred before without insurance update. Ask seller to explain.                                  |
| **Bank NOC** — _"Name of borrower"_ / _"Vehicle owner name"_ field (if car had a loan) | NOC owner name must match RC owner name exactly.                                                                    | 🔴 RED FLAG — NOC for a different person has no legal relevance to this vehicle's RC.                                                                         |
| **Original purchase invoice** — _"Buyer / Customer name"_ field                        | Match only if RC Owner S.No. = 1 (first owner). If S.No. is 2 or higher, invoice name won't match — that is normal. | 🟡 FLAG — Owner S.No. mismatch with invoice when seller claims to be first owner = seller is lying about ownership history.                                   |

## Field 1.2 — Registration Number

**Where to find it:**

- RC Book: Prominently on the cover and inside first page.
- RC Smart Card: Large print on front face.

| Match against                                                                       | What to check                                                                             | Result if mismatch                                                                                                                      |
| ----------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| **Front number plate** on the car                                                   | Must be character-for-character identical — state code, RTO code, series, and all digits. | 🔴 RED FLAG — Wrong vehicle or cloned plates. Do not proceed.                                                                           |
| **Rear number plate** on the car                                                    | Front plate and rear plate must also match each other.                                    | 🔴 RED FLAG — If front and rear don't match each other = one plate is stolen or swapped.                                                |
| **Insurance policy** — _"Registration number"_ / _"Vehicle registration no."_ field | Exact match required.                                                                     | 🔴 RED FLAG — Different number on insurance = this vehicle has no insurance coverage. The policy covers a completely different vehicle. |
| **PUC certificate** — _"Vehicle registration number"_ field                         | Exact match required.                                                                     | 🔴 RED FLAG — PUC for a different registration number is invalid for this vehicle. RTO will reject the transfer.                        |
| **Bank NOC** — _"Registration number of vehicle"_ field (if applicable)             | Exact match required.                                                                     | 🔴 RED FLAG — NOC is for a different vehicle. Useless for this sale.                                                                    |
| **Service history book** — registration number printed on cover or first page       | Should match.                                                                             | 🟡 FLAG — Service book belongs to a different vehicle; maintenance records are irrelevant.                                              |

## Field 1.3 — Chassis Number (VIN)

**Where to find it:**

- RC Book: _"Chassis no."_ field — typically a 17-character alphanumeric code.
- RC Smart Card: Printed below owner name.

| Match against                                                          | What to check                                                                                                                           | Result if mismatch                                                                                                                                                                      |
| ---------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Physical chassis plate** on the car                                  | Locate the stamped chassis number on the car body (see Physical Car section for exact locations). Match all 17 characters. Use a torch. | 🔴 RED FLAG — Any mismatch = possible stolen vehicle with a swapped RC, or a cloned vehicle. Stop immediately. Do not hand over any money.                                              |
| **Original purchase invoice** — _"Chassis number"_ field               | All 17 characters must match RC exactly.                                                                                                | 🔴 RED FLAG — Invoice and RC don't agree = either a registration data-entry error at the time of purchase (can be corrected at RTO), or potential fraud. Investigate before proceeding. |
| **Insurance policy** — _"Chassis number"_ field (if printed on policy) | Should match RC. Not all policies print this, but check if present.                                                                     | 🔴 RED FLAG if present and mismatches.                                                                                                                                                  |
| **Bank NOC** — _"Chassis number"_ field (if present)                   | Should match RC.                                                                                                                        | 🔴 RED FLAG if mismatches.                                                                                                                                                              |

---

## Field 1.4 — Engine Number

**Where to find it:**

- RC Book: _"Engine no."_ field.
- RC Smart Card: Printed below chassis number.

| Match against                                           | What to check                                                                                                                                                 | Result if mismatch                                                                                                                                                                                                                   |
| ------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Physical engine number** stamped on the engine block  | Physically locate the engine number (see Physical Car section). Match every character against RC. Bring a torch — engine numbers are often covered in grease. | 🔴 RED FLAG — If mismatch: engine was replaced. Ask seller for an RTO endorsement showing the engine change was officially recorded. Without endorsement, this is an illegal unrecorded modification and will block the RC transfer. |
| **Original purchase invoice** — _"Engine number"_ field | All characters must match RC exactly.                                                                                                                         | 🔴 RED FLAG if mismatches.                                                                                                                                                                                                           |

## Field 1.5 — Hypothecation / Financier Field

**Where to find it:**

- RC Book: Look for _"Hypothecation in favour of"_ or _"HP"_ or _"Financer name"_ — typically in the endorsement section.
- RC Smart Card: May be printed as _"Hypothecated to [Bank Name]"_ on the front face.
- If blank / "NA" / "NIL" → no hypothecation.

| Match against                                              | What to check                                                                   | Result if mismatch                                                                                                         |
| ---------------------------------------------------------- | ------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| **Bank NOC** — bank name on letterhead / stamp             | If RC shows a bank name, the NOC must be from that exact same bank.             | 🔴 RED FLAG — A NOC from a different bank is completely irrelevant. It cannot be used for HP termination of this RC entry. |
| **Form 35** — _"Name of financier"_ field                  | Financier on Form 35 must match the bank named in the RC hypothecation field.   | 🔴 RED FLAG — Wrong bank on Form 35 = HP termination filing will fail at the RTO.                                          |
| **Insurance policy** — endorsement / hypothecation section | Insurance should also show the same bank as co-insured if RC has hypothecation. | 🟡 FLAG — Inconsistency between RC and insurance hypothecation fields; ask seller to explain which is current.             |

> **Key rule:** If any bank name appears in this field — even if the seller claims the loan is closed — do NOT proceed until you physically see:
> (a) the updated RC with this field blank/NA, **and**
> (b) VAHAN portal also showing no hypothecation.
> Bank NOC + Form 35 alone are not enough — HP termination must be completed and the RC must be updated.

---

## Field 1.6 — Make, Model, Body Type, Fuel Type, Colour

**Where to find it:**

- RC Book: _"Make & Model"_, _"Body type"_, _"Fuel"_, _"Colour"_ fields — typically on the first or second page.

| Match against                                                          | What to check                                                                                              | Result if mismatch                                                                                                          |
| ---------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| **Physical car** — visual inspection                                   | Make and model must match visually. Colour on RC must match actual car colour (check bonnet, doors, boot). | 🔴 RED FLAG — RC says "Swift" but the car badge says "Baleno" = RC belongs to a different vehicle.                          |
| **Insurance policy** — _"Make and model"_ / _"Type of vehicle"_ field  | Make and model on insurance must match RC.                                                                 | 🟡 FLAG — May indicate wrong policy or informal vehicle swap.                                                               |
| **Original invoice** — _"Vehicle description / model / variant"_ field | Invoice model, variant, and colour should match RC entries.                                                | 🟡 FLAG — Invoice variant may differ from RC if dealer registered a different variant by error. Verify.                     |
| **Fuel flap label** on the car — inside the fuel filler flap           | Must match RC _"Fuel"_ field.                                                                              | 🔴 RED FLAG — CNG kit installed but RC says petrol = unregistered illegal modification. Insurance may be void in accidents. |
| **PUC certificate** — _"Fuel type"_ field                              | Must match RC fuel type.                                                                                   | 🔴 RED FLAG — Fuel type inconsistency across documents.                                                                     |

---

## Field 1.7 — Registration Validity / Expiry Date

**Where to find it:**

- RC Book: _"Registration valid upto"_ field.
- RC Smart Card: _"Valid upto"_ printed on face.

| Match against                       | What to check                                                       | Result if mismatch / issue                                                                            |
| ----------------------------------- | ------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| **Today's date**                    | Is the registration currently valid? Is it expiring within 30 days? | 🟡 FLAG — Expired registration = vehicle cannot be legally driven. Factor in re-registration cost.    |
| **Year of manufacture** field on RC | Calculate actual vehicle age vs. what the seller claims.            | 🟡 FLAG — "3-year-old car" with a 2017 registration = 7+ year old vehicle. Seller misrepresented age. |

> **Note:** Private vehicles registered after 2018 receive 20-year initial registration. Older vehicles may have had renewals.

---

## Field 1.8 — Owner Serial Number

**Where to find it:**

- RC Book: _"Owner S.No."_ or similar field — indicates how many times the vehicle has changed registered owners.

| Match against                                 | What to check                                                                                                                                | Result if mismatch                                                           |
| --------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------- |
| Seller's verbal claim about ownership history | If seller says "first owner" but RC shows S.No. = 2 or 3 → seller is lying about ownership.                                                  | 🟡 FLAG — Not a legal deal-breaker but a trust issue. Negotiate accordingly. |
| **Invoice buyer name**                        | If S.No. = 1, invoice buyer name should match RC owner name. If S.No. = 2 or higher, invoice name will differ — that is normal and expected. | Check for consistency, not mismatch per se.                                  |
| **Service history** odometer readings         | High owner count + very low odometer reading = possible odometer tampering. Investigate.                                                     | 🔴 RED FLAG if service entries show higher km than current odometer.         |

## RC Physical Inspection — Red Flags (No Document Comparison Needed)

Look at the physical RC book / smart card itself:

- Any **overwriting, whitener / correction fluid, or scratched-out text** on any field.
- **Pages that appear removed or re-inserted** in the RC book.
- **Lamination torn, cut, or re-applied** on the RC smart card — possible field tampering under the laminate.
- **Ink colour inconsistency** across fields — most fields in blue ink but one field in black ink.
- **RTO seal / stamp** that appears blurry, pixelated, or digitally printed rather than rubber-stamped in ink.
- Smart card that **feels lighter than usual**, has **uneven edges**, or shows any sign of reprinting.
- **Any field where the font style or character size differs** from surrounding fields — indicates overprinting.

# DOCUMENT 2 — SELLER'S IDENTITY PROOF

> Ask for all three if possible. Aadhaar is primary. PAN is mandatory. Passport if available.  
> Ask: _"Please show me your Aadhaar card, PAN card, and one more photo ID."_  
> **Always ask for originals — never accept photocopies for identity verification.**

---

## Field 2.1 — Name on Aadhaar

**Where to find it:** Front face of Aadhaar card — below the photo, in large text.

| Match against                    | What to check                                                                                                 | Result if mismatch                                                             |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| **RC** — _"Name of owner"_ field | Character-for-character identical. This is the primary identity verification step.                            | 🔴 RED FLAG — Any mismatch without a valid legal explanation = do not proceed. |
| **PAN card** — _"Name"_ field    | PAN name must match Aadhaar name (PAN may use initials for middle names — acceptable if family name matches). | 🟡 FLAG — Escalate if family name also differs.                                |

---

## Field 2.2 — Photo on Aadhaar

**Where to find it:** Front face, left side — the photograph.

| Match against                                  | What to check                                                                                                   | Result if mismatch                                                                                                                |
| ---------------------------------------------- | --------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| **The actual person** standing in front of you | Face shape, approximate age, distinguishing features. The person must match the Aadhaar photo beyond any doubt. | 🔴 RED FLAG — Person looks substantially different from Aadhaar photo. Ask for a second ID (Passport / Voter ID) to cross-verify. |
| **PAN card photo**                             | Both Aadhaar and PAN photos should clearly be the same individual.                                              | 🔴 RED FLAG — Photos look like different people = possible document fraud.                                                        |

---

## Field 2.3 — Aadhaar Number (Last 4 Digits)

**Where to find it:** Below the name — 12-digit number (standard Aadhaar masks first 8 digits as XXXX XXXX XXXX).

| Match against                                                      | What to check                                                                                                                                                                                  | Result if mismatch                                                                       |
| ------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------- |
| **Seller's ability to receive OTP** on their Aadhaar-linked mobile | The Parivahan faceless service sends OTP to the Aadhaar-registered mobile. If the seller cannot receive this OTP, the online faceless transfer will fail — offline RTO visit will be required. | 🟡 FLAG — Not a deal-breaker; switches the process from faceless online to physical RTO. |

---

## Field 2.4 — PAN Number

**Where to find it:** Front face of PAN card — 10-character alphanumeric code (format: ABCDE1234F).

| Match against            | What to check                                                                          | Result if issue                                                                                                                                                                 |
| ------------------------ | -------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **4th character of PAN** | P = individual person. C = company. H = Hindu Undivided Family.                        | 🔴 RED FLAG — If buying from an individual, 4th character must be P. A company PAN (C) selling a personal vehicle requires a Board Resolution authorising the sale. Ask for it. |
| **Sale agreement**       | PAN number must be recorded in the sale agreement (Form 29 / Form 30 also require it). | Ensure you copy it correctly.                                                                                                                                                   |

---

## Identity Document Physical Red Flags

- **Lamination that feels too thick**, has air bubbles, or has uneven edges — possible fake ID.
- **Any sign of photo substitution** — different paper texture at the photo area, misaligned photo edges, slight colour difference around the photo border.
- Seller **hesitates or refuses** to show original ID and offers only a photocopy — reject and insist on original.
- PAN and Aadhaar family names are **completely different** with no legal explanation.
- **Hologram / QR code area** on Aadhaar appears scratched, overprinted, or missing — indicates tampered document.

# DOCUMENT 3 — INSURANCE POLICY / CERTIFICATE OF INSURANCE

> Ask for the original policy document or the Certificate of Insurance (usually a 1-page summary).  
> Ask: _"Please give me the original insurance policy document or Certificate of Insurance."_

---

## Field 3.1 — Policy Expiry / Validity Date

**Where to find it:** Certificate of Insurance — _"Policy period"_ or _"Valid from / Valid to"_ fields.

| Match against     | What to check                                            | Result if mismatch / issue                                                                                                                                                         |
| ----------------- | -------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Today's date**  | Is today's date within the valid-from to valid-to range? | 🔴 RED FLAG — Expired insurance = vehicle is uninsured and illegal to drive. You cannot legally test drive, take delivery, or use this vehicle. Seller must renew before handover. |
| Policy type field | Is it third-party only, or comprehensive?                | 🟡 Note — Third-party = bare legal minimum. Comprehensive = better for you post-purchase. Plan your own insurance policy accordingly.                                              |

---

## Field 3.2 — Insured's Name / Proposer Name

**Where to find it:** Policy schedule — _"Name of insured"_ or _"Proposer"_ field.

| Match against                    | What to check                               | Result if mismatch                                                                                                                                                                        |
| -------------------------------- | ------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **RC** — _"Name of owner"_ field | Insurance should be in the RC owner's name. | 🟡 FLAG — Mismatch may indicate an informal transfer where the RC was changed but insurance was not updated. Ask for explanation. Ensure you update insurance to your name post-transfer. |

---

## Field 3.3 — Vehicle Registration Number on Policy

**Where to find it:** Certificate of Insurance — _"Registration no."_ field.

| Match against                               | What to check                | Result if mismatch                                                                                                            |
| ------------------------------------------- | ---------------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| **RC** — registration number                | Exact match.                 | 🔴 RED FLAG — Different number = this policy covers a completely different vehicle. This vehicle has zero insurance coverage. |
| **Front and rear number plates** on the car | All three must be identical. | 🔴 RED FLAG — Any mismatch = uninsured vehicle.                                                                               |

---

## Field 3.4 — Hypothecation / Endorsement Section

**Where to find it:** Policy schedule — look for an _"Endorsement"_ section or _"Hypothecation"_ note showing a bank as co-insured.

| Match against                | What to check                                                                                                                                           | Result if mismatch                                                                                                                                                                                                                    |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **RC** — hypothecation field | If RC shows a bank name, insurance should show the same bank under endorsement. If RC has no hypothecation, insurance endorsement should also be clean. | 🟡 FLAG — If RC shows no hypothecation but insurance shows a bank = either the RC was updated after HP termination but insurance was not informed yet (common), OR the HP termination was not actually done. Verify which is current. |

---

## Field 3.5 — IDV (Insured Declared Value)

**Where to find it:** Policy schedule — _"IDV"_ or _"Sum insured"_ field.

| Match against             | What to check                                                                                               | Note                                                                |
| ------------------------- | ----------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| Seller's **asking price** | IDV = insurer's estimate of current market value. If asking price is significantly above IDV, question why. | Advisory — use as a reference data point in your price negotiation. |

# DOCUMENT 5 — BANK NOC + FORM 35

_(Only required if the vehicle had a car loan)_

> Ask: _"Please give me the original Bank NOC and both signed copies of Form 35."_  
> ⚠️ Both must be originals. Photocopies and PDF printouts are NOT accepted by the RTO.

---

## Field 5.1 — Physical Authenticity of the NOC (Before Reading Any Fields)

**What to check physically — before anything else:**

| Check                                                         | How to verify                                                                                                                                      | Result if fails                                                                                         |
| ------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------- |
| Is it on the bank's official **letterhead**?                  | Bank name, logo, and address must be pre-printed on the paper itself — not typed on a plain white sheet.                                           | 🔴 RED FLAG — Plain white paper with typed bank name = not valid.                                       |
| Does it have an **original ink signature** (not photocopied)? | Run your finger across the signature. Original ink has a slight raised texture. A photocopied signature is completely flat with no texture at all. | 🔴 RED FLAG — Flat signature = photocopy. RTO will reject it.                                           |
| Does it have an **original rubber stamp** of the bank?        | Should show bank's branch name, branch code, and city in ink (blue or red). Should be a physical stamp impression, not a printed logo.             | 🔴 RED FLAG — Printed "stamp" graphic = not a genuine rubber-stamped original.                          |
| Is the **issue date within 90 days** of today?                | Bank NOC is valid for only 90 days from date of issue.                                                                                             | 🔴 RED FLAG — Expired NOC. Seller must obtain a fresh NOC. A ₹500 NOC is worthless if it's 91 days old. |

---

## Field 5.2 — Vehicle Owner Name on NOC

**Where to find it:** _"Name of borrower"_ / _"Name of vehicle owner"_ / _"Account holder name"_ field.

| Match against                         | What to check                                                                        | Result if mismatch                                                                          |
| ------------------------------------- | ------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------- |
| **RC** — _"Name of owner"_ field      | Must be identical.                                                                   | 🔴 RED FLAG — NOC for a different person has no legal relevance to this vehicle's RC entry. |
| **Seller's Aadhaar** — _"Name"_ field | NOC name = RC name = Aadhaar name = the person in front of you. All four must agree. | 🔴 RED FLAG — Any mismatch in this chain = stop.                                            |

---

## Field 5.3 — Vehicle Registration Number on NOC

**Where to find it:** _"Vehicle registration no."_ / _"Registration number"_ field.

| Match against                | What to check      | Result if mismatch                                                          |
| ---------------------------- | ------------------ | --------------------------------------------------------------------------- |
| **RC** — registration number | Must be identical. | 🔴 RED FLAG — NOC is for a different vehicle. Cannot be used for this sale. |

---

## Field 5.4 — Bank Name on NOC vs. RC Hypothecation Field

**Where to find it:** NOC letterhead / rubber stamp bank name vs. RC _"Hypothecation in favour of"_ field.

| Match against                                 | What to check                                                                         | Result if mismatch                                                                                                                                                                                                 |
| --------------------------------------------- | ------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **RC** — _"Hypothecation in favour of"_ field | Bank issuing the NOC must be the exact same bank named in the RC hypothecation entry. | 🔴 RED FLAG — Example: RC shows "HDFC Bank" in hypothecation field → NOC must be from HDFC Bank. A NOC from SBI or Bajaj Finance is completely irrelevant for this RC entry and cannot be used for HP termination. |

---

## Form 35 — Fields to Check

**Where to find it:** A standard RTO form — 2 sets required.

| Field                           | Match against                                                                                    | What to check                                                                                                                           |
| ------------------------------- | ------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------- |
| _"Name of financier"_           | RC hypothecation field                                                                           | Same bank name as in the RC.                                                                                                            |
| _"Vehicle registration number"_ | RC registration number                                                                           | Must be identical.                                                                                                                      |
| _"Name of registered owner"_    | RC owner name + Seller's Aadhaar                                                                 | Must all agree.                                                                                                                         |
| **Signatures**                  | Check: seller (vehicle owner) has signed AND bank's authorised signatory has signed on both sets | Both signatures must be present. A Form 35 signed only by the seller and not by the bank is incomplete and will be rejected by the RTO. |

# DOCUMENT 7 — SERVICE HISTORY BOOK

> The manufacturer-issued service booklet with stamps from authorised service centres.  
> Ask: _"Please share the service history booklet and any service records you have."_

---

## Field 7.1 — Chassis / VIN Number on Service Book Cover

**Where to find it:** Inside front cover of the service booklet — chassis / VIN number printed or stamped.

| Match against                 | What to check       | Result if mismatch                                                                                                   |
| ----------------------------- | ------------------- | -------------------------------------------------------------------------------------------------------------------- |
| **RC** — chassis number field | Must match exactly. | 🟡 FLAG — Service book belongs to a different vehicle. Maintenance records are irrelevant to the car you are buying. |

---

## Field 7.2 — Odometer Readings at Each Service Entry

**Where to find it:** Each service stamp page — _"Km at service"_ or odometer reading recorded.

| Match against                                      | What to check                                                                                                                                                            | Result if mismatch                                                                                                                 |
| -------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------- |
| **Current odometer** on the physical car dashboard | Read each service entry km in chronological order. Each entry must be higher than the previous. The current odometer reading must be higher than the last service entry. | 🔴 RED FLAG — Current odometer lower than any past service entry = odometer was rolled back (tampered). This is fraud. Do not buy. |
| Previous service entry odometer readings           | Numbers must always increase. A service at 45,000 km followed by a service at 38,000 km = odometer was rolled back between these two services.                           | 🔴 RED FLAG — Any decrease in km sequence = tampering.                                                                             |

> **How to calculate normal expected km:**  
> Average Indian car usage: ~12,000–15,000 km per year.  
> Formula: `Expected odometer range = (Vehicle age in years) × 12,000 to 15,000`  
> A 5-year-old car with 20,000 km total (4,000 km/year) is unusually low — either very light use (possible) or odometer tampered (investigate by checking service entries).

---

## Field 7.3 — Dates of Service Entries

**Where to find it:** Each service stamp — date stamped by service centre.

| Match against         | What to check                                                                      | Result if mismatch                                                                                 |
| --------------------- | ---------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| Chronological order   | Dates must always move forward — each service date must be after the previous one. | 🔴 RED FLAG — Dates out of order = forged or backdated stamps.                                     |
| Gaps between services | More than 15–18 months between services for a regularly used car is a flag.        | 🟡 FLAG — Ask seller to explain large gaps. May indicate unserviced period or off-brand servicing. |

---

## Field 7.4 — Service Centre Stamps — Authenticity

**Where to find it:** Each service page — stamps from authorised service centres.

| What to check                                                                                                                                                                                | Result if issue                                                                                      |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- |
| Are all stamps from authorised dealerships / service centres for **this car's brand**? A Maruti stamp on a Hyundai service book = forged.                                                    | 🔴 RED FLAG — Cross-brand stamps = forged service book.                                              |
| Do stamps show natural age-appropriate fading, or are they suspiciously all equally fresh and uniform? Genuine old stamps fade; uniform freshness on entries spanning 5 years is suspicious. | 🟡 FLAG — Possibly backdated forgery to inflate service history for resale.                          |
| Is the handwriting / ink on the km entries consistent, or do some entries look like they were written at the same time?                                                                      | 🟡 FLAG — All entries in identical handwriting and ink = possibly filled in recently in one sitting. |

# PHYSICAL CAR INSPECTION

> These checks are done on the vehicle itself — no documents required. Bring a torch.

---

## Check P.1 — Chassis Number Plate — Location and Tampering Detection

### Where to find the chassis number plate on the car:

| Location                          | Description                                                                                                                                                                         |
| --------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Firewall under the bonnet         | The metal panel between the engine bay and the passenger cabin. Open the bonnet and look at the vertical wall at the rear of the engine bay. Often stamped directly into the metal. |
| Driver-side door jamb (B-pillar)  | Open the front driver's door. Look at the vertical metal post (B-pillar). A sticker with VIN/chassis is often affixed here.                                                         |
| Driver-side floor area            | Lift the floor mat near the accelerator pedal. Some cars have the chassis plate stamped on the floor panel.                                                                         |
| Base of windshield (from outside) | On newer cars, the VIN is laser-etched and visible through the windshield glass at the bottom-left, viewable from outside.                                                          |

### Signs of genuine, untampered chassis plate:

- Stamping is deeply and uniformly embossed into the metal.
- All characters show the same font style and stamp depth.
- Surrounding metal shows the same age and patina as the rest of the car.
- No tool marks, grinding marks, or weld spatter around the plate area.
- No fresh paint over just the plate area while surroundings have aged paint.

### Signs of tampered chassis plate — 🔴 RED FLAG:

- Different font style for some characters vs. others.
- Some characters stamped more deeply or at a different angle.
- Weld marks or grinding marks visible around the plate.
- Plate looks newer than surrounding metal — possible cut-out-and-reattached section.
- Fresh paint specifically over the plate area while surrounding areas have aged paint.
- Double impressions visible on any character (re-stamped over existing numbers).

---

## Check P.2 — Engine Number Location

| Engine type    | Common stamping location                                                 |
| -------------- | ------------------------------------------------------------------------ |
| Petrol engines | Usually on the top of the engine block or on the front face of the block |
| Diesel engines | Often on the side of the block near the oil filter area                  |
| All types      | A flat machined surface on the engine block — not on a sticker           |

> **How to read it:** Use a torch. Wipe the area with a clean cloth to remove grease. The number is stamped / embossed directly into the metal — it is NOT on a sticker. Match every character against the RC.

---

## Check P.3 — Number Plates (HSRP Verification)

### For cars registered after April 2019 in Maharashtra — HSRP is mandatory:

| HSRP Feature                | What to look for                                                                                                             |
| --------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| Chromium hologram sticker   | Top-left of the plate — shows "IND" and Ashoka Chakra. Should have a 3D shimmer when tilted.                                 |
| Laser-etched chassis digits | Last few characters of the chassis number are laser-etched into the plate itself. Match these against the RC chassis number. |
| Snap-lock fasteners         | Plate is attached with non-removable snap-lock studs — cannot be removed with a regular screwdriver.                         |
| Hot-stamped laser code      | A unique code hot-stamped onto the plate surface.                                                                            |

**Match: HSRP laser-etched chassis digits → RC chassis number (last digits must match)**  
🔴 RED FLAG: HSRP digits don't match RC chassis = wrong plate or tampered plate.

---

## Check P.4 — Fuel Flap Label

**Location:** Inside the fuel filler flap — open the fuel cap cover and look inside.

| Match against                         | What to check                                                              | Result if mismatch                                                        |
| ------------------------------------- | -------------------------------------------------------------------------- | ------------------------------------------------------------------------- |
| **RC** — _"Fuel"_ field               | Label says "Petrol only" / "Diesel only" / "CNG". Must match RC fuel type. | 🔴 RED FLAG — Unregistered fuel modification.                             |
| **PUC certificate** — fuel type field | All three (RC, fuel flap, PUC) must agree.                                 | 🔴 RED FLAG — Any disagreement = illegal modification or wrong documents. |

> **CNG kit check:** If you see a CNG cylinder in the boot, a CNG nozzle under the bonnet, or a second fuel gauge on the dashboard, but the RC says "Petrol" — this is an unregistered modification. Insurance may be void in an accident. Ask for the RTO endorsement of the CNG kit installation.

---

## Check P.5 — Paint and Panel Condition (Accident History)

### Where to look:

| Location                         | What to check                                                                                                                                                                                                                                       |
| -------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Panel gaps (bonnet, doors, boot) | Walk around the entire car. All gaps between panels should be uniform. Uneven gaps indicate a replaced or repaired panel.                                                                                                                           |
| Door jamb inner surfaces         | Open every door and look at the inner edge and B-pillar. These surfaces are rarely repainted in accident repairs — original colour is preserved. If inner surface colour differs from the outer panel = that panel was resprayed after an accident. |
| Boot / bonnet inner lip          | Same principle — look at the inner metal surfaces for original vs. touched-up paint.                                                                                                                                                                |
| Window rubber seals              | Paint overspray on rubber seals = car was resprayed without professional masking. Indicates panel repair work.                                                                                                                                      |
| Brake discs                      | Fresh paint dust or overspray on brake discs from a nearby panel repaint = body shop work done.                                                                                                                                                     |

🟡 **FLAG** for any of the above — not necessarily a deal-breaker, but ask the seller for full disclosure of any accident history. Factor into your price negotiation.

---

## Check P.6 — Odometer Reading

**Location:** Dashboard instrument cluster — digital or analogue odometer.

| Match against                                     | What to check                                                                                                                 | Result if mismatch                                                                                                                                                                 |
| ------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Last service entry km** in service history book | Current odometer must be equal to or higher than the last recorded service km.                                                | 🔴 RED FLAG — Current reading lower than a past service entry = odometer rolled back.                                                                                              |
| **All prior service entries** in sequence         | Each service entry km must be higher than the one before it.                                                                  | 🔴 RED FLAG — Any decrease in the service km sequence = odometer was tampered at some point.                                                                                       |
| Expected km range for vehicle's age               | `Vehicle age (years) × 12,000 = low end` / `× 15,000 = high end`. Anything outside 8,000–20,000 km/year is worth questioning. | 🟡 FLAG — Very low km vs. vehicle age warrants investigation. Check for consistent wear on pedals, steering wheel, seat bolster — high-wear cars that show 30,000 km are tampered. |
